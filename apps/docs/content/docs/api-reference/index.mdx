---
title: API Reference
description: Public stable REST endpoints grouped by resource.
---

The reference is generated from the committed OpenAPI contract at `openapi/egdata.openapi.json`.

Use the resource groups in the sidebar to browse endpoints by domain:

| Group | Use |
| --- | --- |
| System | Health and API metadata |
| Regions | Countries and regional pricing helpers |
| Catalog | Cross-catalog discovery endpoints |
| Offers | Epic Games Store offer metadata |
| Offer Details | Offer subresources such as items, media, builds, ratings, and related data |
| Offer Reviews | Read-only public review data for offers |
| Prices | Current prices, price history, and regional comparisons |
| Items | Catalog item details, assets, builds, and changelogs |
| Builds | Verified and migrated manifest health, chronological observations, paginated files, install options, and file-level comparisons |
| Sandboxes | Epic namespace and sandbox resources |
| Free Games | Current and historical free promotions |
| Search | Search, filters, facets, and changelog search |
| Sellers | Seller catalogs, covers, and stats |
| Stats | Global and homepage statistics |

Operation pages include request parameters, request bodies, response schemas, and response examples where the contract has enough detail.
